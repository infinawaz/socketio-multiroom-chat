# Raabta - Realtime Chat

A real-time, premium chat application built with **Node.js, Express, and Socket.io**.
Users can join specific rooms, send messages, see online users, and view chat history.

1. Clone the repository:
   ```bash
   git clone https://github.com/infinawaz/socketio-multiroom-chat.git
   cd socketio-multiroom-chat
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the server:
   ```bash
   npm start
   ```

   ```bash
   npm run dev
   ```

4. Open your browser and go to:
   `http://localhost:3000`
